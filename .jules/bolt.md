## 2024-05-18 - [Optimization of inner loop predictions collection]
**Learning:** In codebases predicting multiple images sequentially with fastai/PyTorch, repeatedly allocating a new pandas DataFrame inside a loop and concatenating them heavily degrades performance. By manually resolving the `max()` values according to a static hierarchy inside the inner loop and collecting simple python dictionaries, memory allocations per item drops significantly. Native Python handles list of dicts creation efficiently, meaning the DataFrame generation overhead is moved from $O(N)$ times inside the loop to $O(1)$ outside the loop.
**Action:** Replace $O(N)$ DataFrame concatenations inside loops with native python `list.append(dict)` operations and instantiate `pd.DataFrame(list_of_dicts)` once at the end of the script to drastically reduce overhead.
